## 📊 Quick Reference

### Data Structure Comparison

| Feature | Stack | Queue | Linked List |
|---------|-------|-------|------------|
| **Order** | LIFO | FIFO | Sequential |
| **Access** | Top only | Front & Back | Any position |
| **Insert** | O(1) | O(1) | O(1) at beginning |
| **Remove** | O(1) | O(1) | O(1) at beginning |
| **Use Case** | Undo, History | Orders, Tasks | Dynamic sequences |

## 📝 Key Concepts

### Stack (LIFO)
- Last element added is first removed
- Used for: undo operations, function calls, expression evaluation
- Operations: `push()`, `pop()`, `peek()`

### Queue (FIFO)
- First element added is first removed
- Used for: task scheduling, order processing, message queues
- Operations: `enqueue()`, `dequeue()`, `peek()`

### Linked List
- Dynamic sequence of nodes connected by pointers
- Used for: dynamic data, efficient insertions/deletions
- Operations: `add()`, `remove()`, `traverse()`
